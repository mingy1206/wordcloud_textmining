<h5>
설정방법

- 글자 폰트와 원하는 워드클라우드의 모양을 선택합니다.
- 원하는 사이트들의 주소를 입력합니다.
- 제외시키고 싶은 단어를 입력합니다.
- 한글 구분이 원할하지 않아 인식을 못할경우 학습시킬 단어를 입력받으면 dictionary에 추가됩니다.
</h5>
